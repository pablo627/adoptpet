# 🐾 AdoptPet – Sistema de Adoção de Animais

O sistema **AdoptPet** é uma aplicação web voltada para facilitar o processo de adoção de animais domésticos, conectando adotantes, cuidadores e abrigos em uma única plataforma.

O objetivo é tornar o processo de adoção mais organizado, acessível e eficiente, permitindo o cadastro, gerenciamento e busca de pets disponíveis para adoção por meio de uma interface intuitiva e responsiva.

---

## 🧰 1. Tecnologias Utilizadas

## Backend

- PHP
- Banco de dados relacional (MySQL)


## Front-end
- HTML5
- CSS3
- JavaScript

## Outras
- XAMPP
  
---

## ⚙️ 2. Funcionalidades Implementadas

- Cadastro e login de usuários  
- Cadastro de animais para adoção  
- Listagem de pets disponíveis  
- Visualização de detalhes dos animais  
- Busca e filtragem de pets  
- Solicitação de adoção  

---

## 🚧 3. Funcionalidades Previstas (Não Implementadas)

- Sistema de notificações avançadas  

---

## 🧩 4. Outras Funcionalidades Implementadas

- Controle de status de adoção dos animais  
- Gerenciamento de dados de usuários e pets  
- Interface responsiva para diferentes dispositivos  
- Organização das informações para facilitar a navegação  

---

## 🧠 5. Principais Desafios e Dificuldades

- Estruturação do fluxo de adoção entre usuários e abrigos  
- Integração entre frontend e backend  
- Modelagem e organização dos dados dos pets  
- Implementação de filtros e busca eficiente  
- Criação de uma interface simples e intuitiva
