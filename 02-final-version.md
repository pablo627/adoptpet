O sistema “AdoptPet” é uma aplicação web voltada para facilitar o processo de adoção de animais domésticos, conectando adotantes, cuidadores e abrigos em uma única plataforma. O objetivo é tornar o processo de adoção mais organizado, acessível e eficiente, permitindo o cadastro, gerenciamento e busca de pets disponíveis para adoção por meio de uma interface intuitiva.

1. Tecnologias utilizadas – Backend e Frontend

Backend: Node.js (ou tecnologia equivalente do projeto), API REST para gerenciamento de usuários e pets, banco de dados relacional (ex: MySQL/PostgreSQL).

Frontend: HTML, CSS e JavaScript, com estrutura responsiva para navegação em diferentes dispositivos.

2. Funcionalidades implementadas
Cadastro e login de usuários;
Cadastro de animais para adoção;
Listagem de pets disponíveis;
Visualização de detalhes dos animais;
Busca e filtragem de pets;
Solicitação de adoção.
3. Funcionalidades previstas e não implementadas
Chat em tempo real entre usuários;
Sistema de geolocalização de abrigos;
Aplicativo mobile nativo;
Sistema de notificações avançadas;
Integração com clínicas veterinárias.
4. Outras funcionalidades implementadas
Controle de status de adoção dos animais;
Gerenciamento de dados de usuários e pets;
Interface responsiva para diferentes dispositivos;
Organização das informações para facilitar a navegação.
5. Principais desafios e dificuldades
Estruturação do fluxo de adoção entre usuários e abrigos;
Integração entre frontend e backend;
Organização e modelagem dos dados dos pets;
Implementação de filtros e busca eficiente;
Garantir uma interface simples e intuitiva.
6. Instruções para instalação e execução
Clonar o repositório:
git clone https://github.com/pablo627/adoptpet
Acessar a pasta do projeto:
cd adoptpet
Instalar dependências do backend:
npm install
Iniciar o servidor:
npm start ou node server.js
Abrir o frontend no navegador (ou rodar via servidor local).
